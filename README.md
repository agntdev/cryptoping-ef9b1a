# CryptoPing

Telegram bot for private crypto watchlists, price-threshold and percent-change alerts, on-demand prices, and optional morning summaries with quiet-hours and suppression logic.

Spec: [`docs/blueprint.md`](docs/blueprint.md).

Built on [agnt-gm.ai](https://agnt-gm.ai). The whole bot is built and refined here as pull requests across successive build passes.
